# Задача: Разработка простого блога с комментариями

## инициализация и запуск
в корне перед стартом хапустить:
`docker compose build`

старт контейнеров:
`docker compose up`

будет запущено два контейнера lr-db и lr-blog, с базой данных и приложением соответственно

## работа сприложением
открывается по ссылке http://127.0.0.1:8080/

доступ к phpMyAdmin: http://127.0.0.1:8080/pma/

при необходимости порт можно изменить в конфиге [docker-compose.yml](docker-compose.yml)

настройки БД приложения находятся в [app/.env](app/.env)

без установки, посмотреть можно [тут](https://blog.pc.madcore.pro/)
